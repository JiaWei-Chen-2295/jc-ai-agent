### 面试深度追问预判

面试官最可能追问的点及准备建议：

1. **"RAG 检索效果怎么评估的？"** → 你已有完整 Benchmark 框架 (Hit@K, MRR, NDCG 等 9 种指标)，跑一次拿到数据即可

3. **"ReAct Agent 死锁怎么检测？"** → `AgentState.hasRepeatedAction()` 检查最近 N 次是否同一工具，超过阈值触发 `FALLBACK`
4. **"多租户怎么做的？有没有考虑数据泄露？"** → ThreadLocal + SQL 强制过滤，Vector Store 自定义实现，所有查询带 `tenant_id`

6. **"为什么自己写 VectorStore 而不用 Spring AI 内置？"** → 需要多租户 `tenant_id` 列级隔离，内置不支持 Compacted conversation
