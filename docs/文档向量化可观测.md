指标含义速查

┌─────────────┬──────┬──────────┬────────────────────────────┐
│    指标     │ 值域 │ 越高越好 │           关注点           │
├─────────────┼──────┼──────────┼────────────────────────────┤
│ Hit Rate@K  │ 0~1  │ 是       │ top-K 里能不能找到答案     │
├─────────────┼──────┼──────────┼────────────────────────────┤
│ MRR         │ 0~1  │ 是       │ 相关结果排多靠前           │
├─────────────┼──────┼──────────┼────────────────────────────┤
│ NDCG@K      │ 0~1  │ 是       │ 排序质量（支持分级相关度） │
├─────────────┼──────┼──────────┼────────────────────────────┤
│ Precision@K │ 0~1  │ 是       │ 返回结果中有多少是对的     │
├─────────────┼──────┼──────────┼────────────────────────────┤
│ Recall@K    │ 0~1  │ 是       │ 所有对的结果找回了多少     │
├─────────────┼──────┼──────────┼────────────────────────────┤
│ P99 延迟    │ ms   │ 否       │ 尾部延迟是否可接受         │
└─────────────┴──────┴──────────┴────────────────────────────┘
